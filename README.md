# JavaScript Digital Clock ⏰

A simple and responsive digital clock built using **JavaScript**, **HTML**, and **CSS**. It displays the current time in hours, minutes, and seconds and updates in real-time.

## Features
- Real-time clock updates every second.
- 12-hour or 24-hour format (can be customized).
- Clean and minimal design with easy-to-read layout.
- Fully responsive for desktops and mobile devices.
